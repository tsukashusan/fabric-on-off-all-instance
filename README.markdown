# fabric-on-off-all-instance

サブスクリプション以下のFabric 容量一括で一時停止・開始するスクリプトです。
[Azure Automation](https://learn.microsoft.com/ja-jp/azure/automation/overview "Azure Automation")の[Runbook](https://learn.microsoft.com/ja-jp/azure/automation/automation-runbook-types?tabs=lps74%2Cpy10#powershell-runbooks "Runbook")に設定することで毎時で設定することが可能です。



## 実行方法

### パラメーター

#### SubscriptionID

自身のサブスクリプションID
#### operation

- suspend</br>
一時停止する場合
- resume</br>
一時停止から開始するばあい
